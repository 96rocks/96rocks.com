Introduce rock960

  • The first 96boards based on rockchip's
  • latest high performace soc
  • RK3399
  • for the community

Design

  • Compact but powerful
  • 4GB ram, 32GB emmc packed into a credit card size body
  • A truely little beast

Rich Interfaces

  • USB 3.1 type C with DP
  • HDMI 2.0 with 4K output
  • 4 lanes M.2 PCI-E 2.1
  • 40 pin low speed expansion header
  • 60 pin high speed expansion header

Open eco-system

  • Unified expansion header and software libraries
  • Multiple venders support
  • Community expansion ideas

Processor

Dual Cortex-A72 + quad Cortex-A53 CPU, up to 2.0GHz frequency. Mali T860MP4 GPU, supports OpenGL ES 1.1 /2.0 /3.0 /3.1 /3.2, Vulkan 1.0, Open CL 1.1 1.2, DX11.

USB 3.0

One USB 3.0 type A port, up to 5.0Gbps. One USB type C port, USB 3.0 with DP 1.2 combined, reversibly plug. One USB 2.0 type A up to 480Mbps.

HDMI

HDMI 2.0, up to 4K x 2K@60HZ, support HDMI CEC and HDCP.

wireless

On board wifi supports 802.11 a/b/g/n/ac 2x2 MIMO, up to 867Mbps, and Bluetooth 4.1 with on board two antennas.

PCIE

On board M.2 key M pcie connector, supports up to 4 lanes PCIE 2.1, max bandwidth 2.0GB, supports M.2 SSD, 4G card and other PCIE cards

Expansion

Standard 96boards high speed expansion header including MIPI/CSI/USB/SDIO/HSIC and low speed expansion header including UART/I2C/SPI/GPIO/I2S.
